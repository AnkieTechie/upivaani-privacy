Privacy Policy

This privacy policy applies to the UPI Vaani – Voice of Every Transaction app or Your All Pay Voice (“Application”), developed by Ankit Kushwaha (“Service Provider”) and provided as a free service. Use of this Application is subject to the terms below.

App Overview

This Application provides voice alerts for UPI transaction messages. It includes a few skippable onboarding screens and requires user acceptance of this Privacy Policy to continue. No registration or login is required.

Data Collection and Use

User-Provided Information

The Application does not require account creation or personal identification. No personally identifiable information is collected from the user.

Automatically Collected Information

Some non-personal technical data such as device type, OS version, and usage behavior may be collected for the sole purpose of improving functionality. This data is not linked to any user identity and is not shared.

Notification Access and Local Storage

The Application reads UPI transaction messages (e.g., sender, amount, date, time) via Notification Access. This data is stored only locally on the user's device using Room Database. No data is uploaded, transmitted, or shared externally.

Auto-Start on Boot

The Application offers an optional setting to start automatically when the device boots. This is disabled by default and can be manually enabled/disabled by the user. No data is collected during this process.

Location Data

The Application does not request, access, or use location data of any kind.

Internet Usage

Internet access is used solely for basic app functions like update checks or error logs. No personal or transactional data is sent or stored online.

Analytics and Ads

The Application does not use any analytics tools or serve ads. It is completely free of third-party tracking or advertising SDKs.

Opt-out Rights

You may stop all collection and use of information by the Application at any time by uninstalling it from your device.

Data Retention

All collected data is stored locally and permanently removed upon uninstallation. No data is stored on any external servers or cloud infrastructure.

Children’s Privacy

The Application is not intended for use by children under the age of 13 and does not knowingly collect any data from them.

Security

All app data is stored on your device. While the Application does not transmit data, users are encouraged to enable device-level security (e.g., screen lock, encryption).

User Consent

By granting Notification Access, you consent to the Application reading your UPI messages for the purpose of providing local voice alerts. This permission can be revoked anytime via system settings.
